# LNFP III sample data

500 MHz Bruker raw data for the milk oligosaccharide LNFP III
(lacto-*N*-fucopentaose III):

```
b-D-Galp-(1-4)-b-D-GlcNAc-(1-3)-b-D-Galp-(1-4)-b-D-Glcp
                     3
                     |
                     1
              a-L-Fucp
```

Six experiments: 1D 1H, three 1H-1H TOCSY (20/40/80ms mixing), 1H-13C HSQC,
and 1H-13C HMBC.

Originally distributed on the EuroCarbDB/CCPN "NMR software" sample CD
(2007), credited to Bas Leeflang. Re-zipped per-experiment (flat, no
`<name>/<expno>/` nesting) for use with nmrium-desktop's Open dialog / Open
Sample menu — confirmed to import correctly across all six experiments.
